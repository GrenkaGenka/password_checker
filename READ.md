# Проверка пароля (psw-checker)

Локальный веб-сервис на Python: настраиваемые правила сложности пароля и визуальный отчёт по каждому правилу. Интерфейс и сообщения — на русском языке.

## Возможности

- **Числовые правила:** минимальная длина, количество цифр, букв, заглавных букв, спецсимволов; ограничение на максимальную длину повтора одного символа подряд (0 — выключено).
- **Запреты (переключатели):** алфавитные и цифровые последовательности (например, `abc`, `123`), раскладочные последовательности на клавиатуре (`qwer`, `qaz` и т.п.), годы вида 1900–2999.
- **API:** `POST /check` с телом JSON — те же поля, что и в форме; ответ: `{ "passed": bool, "rules": [ { "label", "ok" }, ... ] }`.

Зависимости основного приложения — только стандартная библиотека Python (`http.server`, `json`, `re`).

## Требования

- Python 3.10+ (рекомендуется актуальная 3.x)

## Быстрый старт

```bash
cd password_checker
python -m venv venv
```

**Windows (PowerShell):**

```powershell
.\venv\Scripts\Activate.ps1
```

**macOS / Linux:**

```bash
source venv/bin/activate
```

Установка зависимостей для тестов (основной скрипт их не требует):

```bash
pip install -r requirements.txt
```

Запуск сервера:

```bash
python password_checker.py
```

Откройте в браузере: [http://localhost:8080](http://localhost:8080)

Остановка сервера: `Ctrl+C`.

## Тесты (Playwright + pytest)

Тесты в `test_password.py` поднимают браузер и проверяют UI при **уже запущенном** сервере на порту `8080`.

1. Установите браузеры для Playwright (один раз):

   ```bash
   playwright install chromium
   ```

2. В одном терминале запустите `python password_checker.py`.

3. В другом:

   ```bash
   pytest test_password.py -v
   ```

HTML-отчёт (если установлен плагин `pytest-html`):

```bash
pytest test_password.py --html=report.html --self-contained-html
```

В файле тестов есть намеренно «сломанный» сценарий для демонстрации отчёта — см. параметризацию с меткой `broken_test`.
